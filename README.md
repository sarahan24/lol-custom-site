# ⚔️ 롤 내전 관리 사이트

롤(League of Legends) 내전 참가자들의 MT(만남의 광장) 포인트를 관리하고 랭킹을 확인할 수 있는 웹 기반 관리 시스템입니다.

## 🎮 주요 기능

### 1. **MT 랭킹 페이지** 🏆
- 모든 참가자의 MT 포인트를 내림차순으로 정렬하여 표시
- 순위, 닉네임, 티어, MT 포인트 확인 가능
- 메인 페이지에서 "🏆 MT 랭킹" 버튼으로 접속

### 2. **참가자 관리 페이지** ⚙️
- **참가자 추가**: 닉네임, 티어, MT 포인트를 입력하여 새로운 참가자 추가
- **참가자 정보 수정**: 기존 참가자의 정보 편집
- **참가자 삭제**: 등록된 참가자 제거
- 실시간으로 테이블에 반영되며 자동으로 MT 포인트 기준 정렬

### 3. **메인 페이지** 🏠
- 사이트 소개 및 메뉴 네비게이션

### 4. **팀 배치 페이지** ⚔️
- **자동 팀 배치**: 랜덤 섞기 버튼으로 자동으로 팀 배치
- **수동 팀 배치**: 직접 닉네임을 입력하여 팀 구성
- **경기 결과 반영**: A팀/B팀 승리 시 각 팀원의 MT 자동 변동
- 5명씩 2팀으로 구성 (총 10명)
- 참가자 체크박스로 팀 구성원 선택

### 5. **1대1 매칭 페이지** 🥊
- **랜덤 매칭**: 선택된 참가자들을 랜덤으로 1대1 매칭
- **경기 결과 반영**: 승자는 +5, 패자는 -5 MT 변동
- 최대 10개의 매칭 슬롯 제공
- 참가자 선택으로 매칭 참가자 결정

## 💻 기술 스택

### Backend
- **Framework**: Flask (Python 웹 프레임워크)
- **데이터 저장**: JSON 파일 기반 (data.json)
- **동시성**: Threading을 이용한 thread-safe 파일 접근

### Frontend
- **HTML5**: 시맨틱 마크업
- **CSS3**: 반응형 디자인 및 애니메이션
- **JavaScript (ES6+)**: Fetch API를 이용한 비동기 통신

### 기타
- **포트**: 5000 (기본값)
- **문자 인코딩**: UTF-8

## 🚀 설치 및 실행

- Python 3.8 이상
- Flask 설치

### 1단계: 저장소 클론 또는 다운로드
```bash

cd /home/ktw/projects/lol-custom-site
```

### 2단계: 가상 환경 생성 (선택사항이지만 권장)


### 5단계: 브라우저에서 접속
```
http://localhost:5000
또는
http://127.0.0.1:5000
```

## 📖 사용법

### 메인 페이지에서
1. **🏆 MT 랭킹** 버튼: MT 포인트 기준 랭킹 조회
2. **⚙️ 참가자 관리** 버튼: 참가자 관리 페이지 이동
3. **⚔️ 팀짜기** 버튼: 팀 배치 페이지 이동
4. **🥊 1대1 매칭** 버튼: 1대1 매칭 페이지 이동

### 참가자 추가하기
1. "⚙️ 참가자 관리" 페이지로 이동
2. 상단 입력 폼에서:
   - **닉네임**: 참가자의 게임 닉네임 입력 (필수, 중복 불가)
   - **티어**: 드롭다운에서 티어 선택 (기본값: 신튜렁)
     - 선택 가능: 신튜렁, S, A, B, C, D
   - **MT**: 참가자의 현재 MT 포인트 입력 (필수)
3. "참가자 추가" 버튼 클릭
4. 테이블에 자동 추가되고 MT 기준으로 정렬됨
5. 같은 닉네임으로는 중복 등록 불가

### 참가자 정보 수정하기
1. "⚙️ 참가자 관리" 페이지에서 수정할 참가자 행의 "수정" 버튼 클릭
2. 모달 창에서:
   - 닉네임, 티어, MT 포인트 수정
3. "저장" 버튼 클릭
4. 즉시 테이블에 반영

### 참가자 삭제하기
1. "⚙️ 참가자 관리" 페이지에서 삭제할 참가자 행의 "삭제" 버튼 클릭
2. 확인 팝업 창에서 확인
3. 참가자가 데이터베이스에서 제거됨

### MT 랭킹 확인하기
1. "🏆 MT 랭킹" 페이지로 이동
2. 모든 참가자가 MT 포인트 기준 내림차순으로 표시
3. 순위(1, 2, 3...), 닉네임, 티어, MT 포인트 확인 가능

### 팀 배치하기
1. "⚔️ 팀짜기" 페이지로 이동
2. 상단에서 참가자 선택:
   - 등록된 모든 참가자가 체크박스와 함께 표시
   - 게임에 참가할 선수를 체크박스로 선택 (기본: 전체 선택)
   - 최소 10명 이상 선택 필요
3. 두 가지 방법 중 선택:
   - **랜덤 배치**: "🔀 랜덤으로 섞기" 버튼을 클릭하면 선택된 참가자 중 10명을 자동으로 5명씩 2팀으로 배분
   - **수동 배치**: A팀과 B팀 칸에 직접 닉네임 입력
4. 경기 결과 반영:
   - A팀이 이겼을 때: "A팀 MT 변동" 칸에 점수 변동값 입력 후 "적용" 버튼 클릭
   - B팀이 이겼을 때: "B팀 MT 변동" 칸에 점수 변동값 입력 후 "적용" 버튼 클릭
   - 입력된 팀원들의 MT 자동 업데이트
5. **신규 참가자 자동 생성**:
   - 팀에 입력된 닉네임이 등록되지 않은 참가자면 자동으로 MT 0으로 생성 후 변동값 반영

### 1대1 매칭하기
1. "🥊 1대1 매칭" 페이지로 이동
2. 상단에서 참가자 선택:
   - 등록된 모든 참가자가 체크박스와 함께 표시
   - 매칭에 참가할 선수를 체크박스로 선택
3. 매칭 방법:
   - **랜덤 매칭**: "🔀 랜덤 매칭" 버튼을 클릭하면 선택된 참가자들을 랜덤으로 1대1 매칭
   - **수동 매칭**: 각 매칭 슬롯에 선수 1, 선수 2 닉네임 직접 입력
4. 경기 결과 반영:
   - "선수 1 승리" 또는 "선수 2 승리" 버튼 클릭
   - 승자: +5 MT, 패자: -5 MT 자동 반영
5. **초기화**: "🗑️ 초기화" 버튼으로 모든 매칭 데이터 초기화
6. **신규 참가자 자동 생성**:
   - 미등록 닉네임 입력 시 자동으로 참가자 생성 후 MT 변동 반영

## 🗂️ 데이터 구조

### data.json 파일 형식
```json
[
  {
    "nickname": "신튜렁",
    "tier": "신튜렁",
    "mt": 1520
  },
  {
    "nickname": "미드괴물",
    "tier": "S",
    "mt": 1490
  }
]
```

### 필드 설명
| 필드명 | 타입 | 설명 |
|--------|------|------|
| nickname | string | 참가자의 게임 닉네임 |
| tier | string | 참가자의 티어 (신튜렁, S, A, B, C, D) |
| mt | number | 참가자의 MT 포인트 (정수) |

## ⚠️ 주의사항

1. **데이터 저장**: 모든 참가자 데이터는 `data.json` 파일에 저장되며, 서버 종료 후에도 유지됩니다.

2. **동시성 처리**: 여러 요청이 동시에 들어올 경우 thread lock을 통해 데이터 무결성이 보장됩니다.

3. **입력 검증**: 
   - 닉네임과 MT 포인트는 필수 입력값입니다.
   - 닉네임은 중복 등록 불가 (같은 닉네임 존재 시 오류 메시지)
   - MT 포인트는 0 이상의 정수여야 합니다.

4. **XSS 방지**: 모든 사용자 입력은 HTML escape 처리되어 보안이 보장됩니다.

5. **인덱스 기반 관리**: API는 배열의 인덱스를 기반으로 작동하므로, 참가자 삭제 후 인덱스가 변경될 수 있습니다.

## 🐛 문제 해결

### 포트 5000이 이미 사용 중인 경우
```bash
python3 main.py --port 5001
```

### 페이지가 로드되지 않는 경우
1. Flask 서버가 실행 중인지 확인: 터미널에서 `python3 main.py` 실행
2. 브라우저 캐시 삭제 후 새로고침 (Ctrl+F5)
3. 포트 번호 확인: 기본값은 5000

### 참가자 추가/수정 시 오류
1. 닉네임을 입력했는지 확인
2. MT 포인트가 숫자인지 확인
3. 브라우저 콘솔에서 에러 메시지 확인
